Trabalho Desenvolvimento Web III - Rotas

Participantes: Ayrton Senna Cabani Bastos

Breve descrição:

O trabalho foi realizado usando Node.js e o framework Express.js, para pratica do uso de ROTAS.
o trabalho consiste em uma aplicação na qual fornece 5 telas sendo elas:

1 -> O index
2 -> Uma tela de exibição do conteudo fornecido
3 -> Uma tela de exibição dos links que o conteudo possui
4 -> Uma tela de exibição dizendo se o arquivo foi encontrado
5 -> Caso o caminho na URL seja invalido

Para acessar o conteudo:

Para exibilção do conteudo na URL digite: /entrada

localhost:4250/entrada

Para exibilção dos links na URL digite: /links

localhost:4250/links

Para verificar na URL digite: /validar

localhost:4250/validar
